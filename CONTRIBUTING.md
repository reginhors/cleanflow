# Contributing to Smart Cleaning Operations

Thank you for your interest in contributing to **Smart Cleaning Operations**.

This project is a concept prototype created for the GRAILS competition. Contributions are welcome, especially when they improve clarity, usability, documentation or responsible AI design.

## How to contribute

### 1. Report an issue

Open an issue for:
- bugs,
- unclear documentation,
- accessibility problems,
- feature suggestions,
- or responsible AI / privacy concerns.

Please describe:
- what you found,
- what you expected,
- and, where relevant, how the issue can be reproduced.

### 2. Propose a change

For code or documentation changes:
1. create a separate branch,
2. make focused changes,
3. use clear commit messages,
4. and open a pull request with a short explanation of what changed and why.

### 3. Keep contributions aligned with the project principles

Contributions should support the core design principle:

> **AI should support cleaners, not monitor them.**

Please avoid proposals that introduce:
- GPS tracking,
- background employee monitoring,
- individual ranking by default,
- or automated decisions without human oversight.

## Development notes

The current prototype focuses on the mobile PWA and uses structured dummy data.  
Future implementation ideas should be clearly marked as concept, pilot or production work.

## Code of conduct

Please communicate respectfully and constructively.  
The project should remain open, collaborative and focused on responsible AI.
